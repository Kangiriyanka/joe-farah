## Zero-Based Numbering

<br/>
### Musical Array

<br/>

Given an index <b class="">i</b> in <a class="secondary-a "href= "https://en.wikipedia.org/wiki/Zero-based_numbering">zero-based numbering</a>, you can find its corresponding entry by starting the count from 0, but I prefer to pinpoint that entry by counting the i elements before it. For example, the index of F is 5, because there are 5 elements before it. If you're into music, you can shift the array left and right with the arrows to change the root key.
