## 効率は自分で意味を決める

<br/>

効率に関する本をたくさん読んでから、私にとって効率はどんな意味だろうと考え始めました。効率は色んな意味がありますなので、簡単な質問で定義させてください：


<div class="m-auto text-center my-2 p-3">
<p class="bold-rounded"> "環境<span class="text-emerald-600"> X</span> で仕事<span class="text-emerald-600"> Y</span> に集中すれば、どんな成果 <span class="text-emerald-600"> Z </span>をもたらすか？"</p>
</div>

以降の質問の答えは数え切れないです。まず、みんなの働ける環境 <span class="text-emerald-600">X</span> が異なります。例えば、静かなところが好きな人は、うるさい環境で作業することが難しいです。そのほか、仕事 <span class="text-emerald-600">Y</span> の種類によって適切な集中が必要です。自分のレベル以上の仕事のハードルが高くなったら始まりづらい上に続けるのも大変です。最後に、働いた分にどんな成果を期待しているかと問い合わせれます: "私が<span class="text-emerald-600"> Y</span> をしたら、なにが得られるでしょう？"

<br/>
正直に、条件がありすぎますから、効率の意味は自分なりに答えを見つければ良いと思います。以降の質問の答えはなんですか？


<br/>

## 私のこたえ

<br/>

私にとって効率の意味は場所と時間を決めて、心の余裕を持って作業しながら成長することです。立派なスケジュールを立てるよりも、自分を批判せずやることを続けます。個人的に、度々自分に厳しいですから、困った時はさらに集中が切れやすくなります。
そういう性格なので、心の余裕を重視して、能力と効率を意識しないとします。プログラミングのバッグやギターコードがうまくいかない時は、責める姿勢をとらないようとします。比喩的に、自分に期待しすぎると、ネガティブな気持ちが重さとなり、結局仕事を焦るんです。

<br/>

仕事のペースに意識することも大事です。もちろん、計画を作り時間を計ることがいいと思います。しかし、プロセスに詰まってプレシャーが感じる場合、どうやって応変しますか？ どのタイミングで手伝いを求めますか？ 時々、もうちょっと頑張って解決するより休憩した方が賢い状況もあります。最終的に、自分の判断を信じて、もうちょっと続けるかそれともやめるかと選択をします。

<br/>
## あなたの方は？
<br/>

 たくさんのやり方を試した後、素直に判断して相応しい方法を続けることが有効だと信じています。
もちろん、一貫した習慣を保つのは決して簡単ではないです。なぜかというと日常で起こることが気持ちも日の流れを影響するからです。大事なのは効率にはまらず自分が習ったことを自分の世界に活かせることが好ましいです。
