The first step is transforming our initial flat array into a structure we can easily perform lookups in. We could simply choose to work with what we have, the 1D array, but any search of a specific category would require a scan of the whole array.
We can optimize the search by recursively bundling up the categories into parent-children entries.

```python
def _build_category_tree(categories):
    children_map = defaultdict(list)

    # SETUP
    for cat in categories:
        children_map[cat.parent_id].append(cat)
   

    def dfs(parent_id):

        if parent_id not in children_map:
            return []
        return [
            {
                **cat.to_json(),
                "children": dfs(cat.id)
            }
            for cat in children_map[parent_id]
        ]

    # Top level categories exist in the None bucket
    return dfs(None)


```
